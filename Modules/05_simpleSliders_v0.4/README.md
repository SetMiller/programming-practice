# Работа с классами на примере слайдера

Интересные моменты:  
- создание класса **`Slider`**
- импорт класса
- передача настроек классу ввиде объекта
  ```javascript
    {
      btnNext: '.gallery-3 .buttons .next',
      btnPrev: '.gallery-3 .buttons .prev',
      images: '.gallery-3 .slider img',
      auto: true,
      mouseBlock: true,
    }
  ```
